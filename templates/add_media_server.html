Finding Your Plex Token
  1. Sign in to Plex web app
  2. Open Developer Tools (F12)
  3. Go to the Network tab
  4. Refresh the page
  5. Look for a request with "X-Plex-Token=" in the URL
  6. The token is the value after "X-Plex-Token="
